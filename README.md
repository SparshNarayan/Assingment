🏏 International T20 Cricket Data Analysis
📌 Project Overview

This project presents an in-depth analysis of International T20 cricket match data using Python data science libraries. The objective is to extract meaningful insights by performing data cleaning, exploratory data analysis (EDA), statistical computations, and visualizations.

The analysis is implemented in a Jupyter Notebook (Google Colab), making it easy to understand and reproduce.

🎯 Objectives
Perform data cleaning and preprocessing
Conduct exploratory data analysis (EDA)
Apply statistical methods using NumPy
Identify trends and patterns in T20 matches
Create insightful visualizations
🗂 Dataset

File: International.csv

The dataset contains:

Match venues and cities
Match results and winning teams
Run margins and wicket margins
Toss decisions
Match dates
🛠 Technologies Used
Python – Programming language
Pandas – Data manipulation
NumPy – Numerical analysis
Matplotlib – Data visualization
Seaborn – Statistical visualization
Google Colab – Development environment
📊 Analysis Workflow
1️⃣ Data Loading
Imported dataset using Pandas
2️⃣ Data Exploration
Checked dataset dimensions
Inspected column names and data types
Generated summary statistics
3️⃣ Data Cleaning
Handled missing values
Removed duplicate entries
Renamed columns for clarity
Created new derived features
4️⃣ Statistical Analysis

Used NumPy to calculate:

Mean
Median
Standard deviation
Variance
Maximum and minimum values
5️⃣ Data Aggregation

Performed operations using:

groupby()
value_counts()
pivot_table()
Sorting and filtering
6️⃣ Data Visualization

Created visualizations such as:

Bar charts
Histograms
Pie charts
Count plots
Box plots
Heatmaps

These help in understanding team performance and match trends effectively.
